## 🧑🏻‍💻 구현 목표

- [x] 시간을 분으로, 분을를 시간으로 변환
- [x] km를 m로, m를 mk로 변환
- [x] 섭씨를 화씨로, 화씨를 섭씨로 변환
- [x] kg를 파운드로, 파운드를 kg로 변환
